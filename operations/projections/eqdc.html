Equidistant Conic¶

Classification

Conic

Available forms

Forward and inverse, ellipsoidal

Defined area

Global

Alias

eqdc

Domain

2D

Input type

Geodetic coordinates

Output type

Projected coordinates

proj-string: +proj=eqdc +lat_1=55 +lat_2=60¶

Parameters¶

Required¶

+lat_1=<value>¶

First standard parallel.

Defaults to 0.0.

+lat_2=<value>¶

Second standard parallel.

Defaults to 0.0.

Optional¶

+lon_0=<value>¶

Longitude of projection center.

Defaults to 0.0.

+ellps=<value>¶

See proj -le for a list of available ellipsoids.

Defaults to “GRS80”.

+R=<value>¶

Radius of the sphere given in meters. If used in conjunction with +ellps +R takes precedence.

+x_0=<value>¶

False easting.

Defaults to 0.0.

+y_0=<value>¶

False northing.

Defaults to 0.0.
